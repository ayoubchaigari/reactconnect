# ReactConnect

Mini réseau social en ReactJS utilisant JSONPlaceholder API.

## Fonctionnalités

- Liste des utilisateurs
- Affichage des posts par utilisateur
- Commentaires par post
- Ajout de nouveaux posts
- Design responsive en CSS pur

## Technologies

- ReactJS
- Axios
- Fetch API
- useState / useEffect
- CSS (sans Bootstrap)
